# fizzbuzz

Build FizzBuzz: given a positive integer N, print the numbers 1..N, one per
line, replacing multiples of 3 with `Fizz`, multiples of 5 with `Buzz`, and
multiples of both with `FizzBuzz`. Non-integer or non-positive input prints
an error.

When done, verify it works and print `PASS` on the last line. Otherwise
print a short reason.
